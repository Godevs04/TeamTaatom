# Collections Feature - Usage Guide

## Overview
Collections allow you to organize your posts into themed groups, making it easier to find and share related content.

## How to Access Collections

### From Profile Page:
1. Go to the **Profile** tab (bottom navigation)
2. Scroll down to find the **Collections** section
3. Tap on the Collections card to view all your collections

### From Post Menu:
1. Open any post in your feed
2. Tap the **three-dot menu** (⋮) in the top right
3. Select **"Add to Collection"**
4. Choose a collection from the list

## Creating a Collection

1. **From Collections Page:**
   - Tap the **"+"** button in the top right corner
   - OR tap **"Create Collection"** if you have no collections

2. **Fill in Collection Details:**
   - **Name** (required): Enter a name for your collection (max 50 characters)
   - **Description** (optional): Add a description (max 200 characters)
   - **Public Toggle**: 
     - **ON** (Blue): Anyone can view this collection
     - **OFF** (Gray): Only you can view this collection

3. **Save:** Tap **"Save"** in the top right to create the collection

## Adding Posts to Collections

### Method 1: From Post Menu
1. Open any post
2. Tap the three-dot menu (⋮)
3. Select **"Add to Collection"**
4. Choose a collection from the list
5. The post will be added instantly

### Method 2: From Collection Detail Page
1. Open a collection
2. Tap **"Add Posts"** button
3. Browse and select posts to add
4. Tap **"Done"** to save

## Managing Collections

### Viewing Collections:
- Tap on any collection card to see all posts in that collection
- View collection details, description, and post count

### Editing Collections:
1. Open a collection
2. Tap the **edit icon** (pencil)
3. Modify name, description, or privacy settings
4. Save changes

### Deleting Collections:
1. Open a collection
2. Tap the **three-dot menu**
3. Select **"Delete Collection"**
4. Confirm deletion

### Removing Posts from Collections:
1. Open a collection
2. Find the post you want to remove
3. Tap the post to open options
4. Select **"Remove from Collection"**

## Collection Features

### Public Collections:
- Visible to all users
- Can be shared via link
- Appear in search results
- Great for showcasing your best work

### Private Collections:
- Only visible to you
- Perfect for personal organization
- Not searchable by others

### Collection Cover Image:
- Automatically uses the first post's image
- Can be manually set from collection settings

## Tips for Using Collections

1. **Organize by Theme:**
   - Create collections like "Travel Adventures", "Food Diary", "Workout Progress"

2. **Use Descriptive Names:**
   - Clear names make collections easy to find later

3. **Keep Collections Focused:**
   - Don't add too many unrelated posts to one collection

4. **Public vs Private:**
   - Use public collections for content you want to share
   - Use private collections for personal organization

5. **Regular Maintenance:**
   - Review collections periodically
   - Remove posts that no longer fit
   - Update descriptions as needed

## Activity Feed

### Accessing Activity Feed:
1. Go to **Profile** tab
2. Scroll to find **"Activity Feed"** section
3. Tap to view activities

### What You'll See:
- **Post Created**: When friends create new posts
- **Post Liked**: When friends like posts
- **Comment Added**: When friends comment on posts
- **User Followed**: When friends follow new users
- **Collection Created**: When friends create collections
- **Mentions**: When you're mentioned in posts/comments

### Filtering Activities:
- Use filter buttons to view specific activity types
- Filter by: All, Posts, Likes, Comments, Follows, Collections, Mentions

### Privacy:
- Only see activities from users you follow
- Your activities are visible to your followers
- Control visibility in Settings > Privacy

## Troubleshooting

### Can't see Collections section?
- Make sure you're logged in
- Refresh the profile page (pull down)

### Can't add post to collection?
- Check if you have at least one collection created
- Create a collection first, then try adding posts

### Collection not showing?
- Check if collection is set to private
- Verify you're viewing your own collections

### Activity Feed empty?
- Make sure you're following other users
- Check privacy settings
- Activities only show from users you follow
