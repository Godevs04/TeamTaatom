# Agent Instructions

## Workflow

### Step 1: Understand Project

- Read context.md
- Scan entire codebase
- Explain architecture before coding

---

### Step 2: Bug Fixing

- Fix bugs one at a time
- Explain root cause
- Apply minimal fix

---

### Step 3: Feature Development

- Build features in isolation
- Reuse existing components
- Avoid modifying core logic

---

## Output Format

Always provide:

1. Understanding
2. Plan
3. Changes
4. Code

---

## Priority

1. Do not break existing code
2. Fix bugs
3. Add features
